Hexo框架(一):个人博客的搭建

发表于 2019-11-10 阅读次数：

前言

搭博客需要一个理由.关于为什么要自己搭建一个博客,我想是在看到别人清晰有条理的布局与归类后,为了改善自己归纳整理的能力,于是决定自己搭一个
(其实更多是冲着看板娘去的).捣鼓了一个下午之后,终于搞清楚一点门路.就以此记录一点心得吧

搭建流程

搭建个人博客=域名(外网可访问的地址)+外网服务器(外网存储文件图片的地方)+解析(本地与服务器做映射)

搭建流程大概分为:

  1. 下载git,下载node.js,搭建Hexo

  2. GitHub注册与部署

  3. 域名购买

  4. 域名解析

环境的下载与安装

下载并安装git：https://git-scm.com/download/win

下载并安装node.js ；https://nodejs.org/en/download/current/

当上面两个都下载安装完毕后,搭建Hexo框架

①先在本地磁盘中建立一个Hexo文件夹用于存储本地文件

②进入文件右键点击空白处，选中Git Bash Here，依次输入以下命令：

a. 安装Hexo：npm install -g hexo-cli

b. 初始化Hexo: hexo init

c. 启动Hexo服务: npm install -->hexo server -->hexo clean -->hexo generate -->hexo deploy

服务器的部署

注册/登录github：https://github.com/，然后新建一个仓库(借用github服务器)。仓库名字为用户名 + .github.io。

将Hexo文件夹上传至仓库中。

购买域名

去阿里云购买一个域名www.aliyun.com

域名解析

购买成功进行备案后。在阿里云的管理控制台进行域名解析
